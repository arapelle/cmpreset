# Concept

Python script which manage CMakeUserPresets.json files.

# Requirements

Binaries:

- CMake 3.19 or later
- git
- python 3.6

# Install

`pip3 install cmpreset`

# How to use it

## cmpreset configure
## cmpreset link
## cmpreset copy
## cmpreset save
## cmpreset push
## cmpreset pull
